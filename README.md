# 摸鱼人倒计时

> 一个谷歌浏览器倒计时插件

## 安装步骤

```bash
# 安装依赖
npm install

# 打包
npm run build
```

打开谷歌浏览器
输入链接[chrome://extensions/](chrome://extensions/)
将打包生成的 dist 文件夹拖进去

## 插件默认值

1. 考勤时间默认为 9:00-18:00
2. 发薪日默认为每月 10 号
3. 工作制默认为双休，暂不可修改
