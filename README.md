# 摸鱼人倒计时

> 一个谷歌浏览器倒计时插件

## 安装步骤

注意：node 版本要求 14.x

```bash
# 安装依赖
npm install

# 打包
npm run build
```

在谷歌浏览器中打开链接 chrome://extensions/

打开开发者模式，加载已解压的扩展程序

选中打包生成的 dist 文件夹

## 插件默认值

1. 考勤时间默认为 9:00-18:00
2. 发薪日默认为每月 10 号
3. 工作制默认为双休，暂不可修改
4. 点餐提醒默认值为空，也就是不提醒
5. 法定节假日信息需要在 src/holiday.js 文件中配置
